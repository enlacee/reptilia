## Nomenclatura
Prefijos para manejar con relativo orden el sistema de carga.

CAMPAÑAS:

	70 -> Pruebas QAS
	80 -> MiBanco
	90 -> Movistar

LISTAS: `80270820`

	80 -> codigo de campaña
	27 -> dia
	08 -> mes
	20 -> año
